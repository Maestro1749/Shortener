# Shortener

REST API сервис для сокращения ссылок, написанный на Go с использованием слоистой архитектуры.

## Функционал
* Создание короткой ссылки
* Редирект на оргинальный URL
* Обработка дубликатов (одна ссылка - одна короткая ссылка)
* Graceful shutdown сервера

## Стек
* Go
* PostgreSQL
* REST API
* zap (логирование)

## Архитекура
Проект основан на слоистой архитектуре:
**handler -> service -> repository**
**Handler** - работа с HTTP
**Service** - бизнес-логика
**Repository** - работа с БД
**Models** - структуры данных

## API
### Создать короткую ссылку
POST /shorten
{
    "link": "https://example.com"
}
Ответ:
{
    "short_url": "abc123"
}

### Редирект
GET /{short}
Перенаправляет на оригинальный URL
